1,0 - Enable / Disable Debug mode

With Debug mode on,
	IJKLOU - point lights movement, where available
	F1,F2 - toggle front face culling
	F3,F4 - toggle wireframe mode
	F5,F6 - toggle light on/off
	F7,F8,F9 - toggle point, directional, spotlight
	AXES, LIGHTBALLS and debugging TEXT won't be rendered
