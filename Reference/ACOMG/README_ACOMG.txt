WASD - move forward, left, backward, right
Arrow keys / mouse - pitch, yaw camera
Left mouse button - change radius around camera on platform to different texture
1,0 - Debug Mode On; Show / Hide Axes and text
B(hold) - disable bloom & tone-mapping
N(hold) - show render passes

With Debug mode on,
	F1,F2 - toggle front face culling
	F3,F4 - toggle wireframe mode
